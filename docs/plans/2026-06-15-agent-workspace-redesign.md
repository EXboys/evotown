## 左栏 (256px)

### Agent 信息卡 (固定不变)
- 头像 emoji + 名称
- workspace ID (截断)
- 状态 badge

### 三段可折叠区域（同一时间仅一个展开）

| 区域 | 收起(3行) | 展开 |
|------|----------|------|
| 💬 对话历史 | 最近3条 + 新对话按钮 | 完整列表，撑满剩余高度 |
| 🧩 已下发技能 | 最近2条 + 数量 | 全部技能详情，撑满 |
| 📚 知识库 | 最近3条 + 数量 | 全部知识库条目，撑满 |

### 折叠规则
- 按钮文字："展开" / "收起"
- 点击任意 "展开" → 该区域展开，其他自动缩为 3 行
- 默认展开对话历史

### 左栏折叠
- 顶部 [◀] 收起 → 左栏消失，中栏左侧边缘浮现 [▶] 圆形按钮
- 点击 [▶] → 左栏恢复

## 中栏 (flex-1)

### 顶部
- 无 Agent 信息（左侧已有）
- 无模型/Skills 选择器

### 对话线程 (无 padding, 顶格)

| 消息类型 | 宽度 | 对齐 |
|---------|------|------|
| 用户消息 | max-w-[75%] | 右 |
| Agent 文字/Markdown | max-w-[75%] | 左 |
| Agent 代码块 | max-w-[75%] | 左 |
| Agent 图片/视频 | max-w-[75%] | 左 |
| HTML iframe 预览 | **全宽** | 左 |
| 执行日志 (fold) | max-w-[75%] | 左 |
| 事件时间线 (fold) | max-w-[75%] | 左 |
| 文件下载 (fold) | max-w-[75%] | 左 |
| 状态提示 | 居中 | - |

### 输入区 (顶格)
- textarea（Enter 发送 · Shift+Enter 换行）
- 📎 附件上传
- [发送 →] 按钮
- 无模型选择器（管理员配置）
- 无 Skills 选择器（左侧已展示）

## 右栏 (288px)

### 工作目录
- 仅在 Agent 执行开发任务（MCP/Skill/代码）时显示
- 无开发任务 → 不渲染右栏，中栏全宽
- 目录树懒加载：点击 [▸] 才请求子节点
- 右侧 [▶] 收起按钮

## 改动范围

纯前端，零后端改动：

| 改动 | 类型 |
|------|------|
| 三栏宽度调整 | CSS |
| 左右栏折叠 | React state |
| 左栏三段折叠互斥 | React state |
| 技能/知识库展示 | 调已有 API (只读) |
| 中栏去顶栏 + 对话顶格 | 删 JSX |
| 输入区去模型/Skills | 删 JSX |
| HTML 全宽 | CSS |
| 右栏工作目录 | 已有 API |
