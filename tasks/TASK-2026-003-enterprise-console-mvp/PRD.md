# PRD: Enterprise console frontend MVP

## Problem

The backend can ingest external engine registrations and completed runs, but users can only inspect that data with API calls. Evotown needs a visible console slice to make the enterprise control plane direction tangible.

## Goals

- Show registered engines and external runs in the web UI.
- Provide summary cards that make the ingest data understandable at a glance.
- Keep the existing Arena experience intact.
- Add clear navigation from the landing page and Arena to the console.

## Non-goals

- Full enterprise navigation shell.
- Policy Center UI.
- Asset Registry UI.
- Inline engine registration forms.
- Editing or deleting engine/run records.

## Target users

- Evotown maintainers testing the ingest MVP.
- Enterprise evaluators looking for control-plane direction.
- Runtime integrators checking whether their reports reached Evotown.
