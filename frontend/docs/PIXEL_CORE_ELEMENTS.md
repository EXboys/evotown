## 一、NES 受限调色板

所有颜色必须从以下调色板中选取，禁止使用渐变、半透明、抗锯齿。

```
核心色板:
  亮绿 (草地基底):    #30B830
  暗绿 (草地斑点):    #207020
  深绿 (树木主体):    #186818
  极深绿 (树木暗面):  #0C400C
  浅棕 (道路基底):    #D0A868
  中棕 (道路斑点):    #906830
  深棕 (岩石/树干):   #604010
  岩棕 (山脚岩石):    #885820
  深灰 (山体基底):    #585858
  中灰 (山体中调):    #888888
  浅灰 (山体高光):    #B8B8B8
  白色 (山顶/高光):   #F8F8F8
  纯黑 (轮廓线):      #000000
  蓝色 (角色头盔):    #3060F0
  肤色 (角色面部):    #F0B088
  铠甲白 (角色身体):  #F0F0F0
```

## 二、草地/平原 (Grassland)

### 视觉特征
- **纯色填充** + **规则抖动斑点**，无渐变
- 斑点为 1px 的深绿色点，均匀分布在亮绿底色上
- 斑点间距约 2px，水平和垂直方向交错排列（棋盘式抖动）
- 每个 16×16 Tile 内斑点布局一致，无缝拼接

### 颜色
| 层 | 颜色 | 用途 |
|----|------|------|
| 基底 | #30B830 | 整个 Tile 填充 |
| 抖动点 | #207020 | 1px 斑点，间隔2px |

### 生成规则 (Canvas/程序化纹理)
```
对每个像素 (x, y):
  if (x + y) % 3 == 0:
    color = #207020  // 暗绿斑点
  else:
    color = #30B830  // 亮绿基底
```

## 三、道路 (Road/Path)

### 视觉特征
- 浅棕/沙土色基底 + 深棕色抖动斑点（与草地同一抖动方式）
- 道路边缘为**不规则有机曲线**，非直线，与草地自然过渡
- 过渡区域：道路边缘有 2-4px 宽的混合带，草色和道路色交替出现
- 路面纹理密度与草地一致

### 颜色
| 层 | 颜色 | 用途 |
|----|------|------|
| 基底 | #D0A868 | 道路主体填充 |
| 抖动点 | #906830 | 1px 斑点，间隔2px |
| 边缘过渡 | #30B830 / #D0A868 交替 | 锯齿边缘 |

### 边缘处理
```
道路边缘不使用平滑曲线，而是像素级锯齿：
  ████████
 ██████████
████████████    ← 每行增减 1-2px 宽度
████████████
 ██████████
  ████████
```

## 四、山脉 (Mountains)

### 视觉特征
- 由多个**重叠的尖三角锥**组成，密集排列
- 每个山峰为等腰三角形，顶部尖锐，基部宽约 8-12px，高约 12-16px
- 三色分层渲染：深灰基底 → 中灰中调 → 白色峰顶高光
- 每个山峰有 1px 黑色轮廓线
- 山峰前后层叠，前排遮挡后排，创造纵深感
- 山脚处散落棕色圆形岩石/丘陵作为过渡

### 颜色
| 层 | 颜色 | 用途 |
|----|------|------|
| 轮廓 | #000000 | 1px 黑色描边 |
| 暗面 | #585858 | 山体左侧/底部 |
| 中调 | #888888 | 山体右侧/中部 |
| 高光 | #F8F8F8 | 山顶 2-3px 尖端 |

### 单个山峰结构 (约 10×14px)
```
     ██         ← 白色峰顶 (#F8F8F8)
    ████
   ██░░██       ← 中灰面 (#888888) 
  ██░░░░██
 ██▓▓░░░░██     ← 深灰面 (#585858) 在左侧
██▓▓▓▓░░░░██
████████████    ← 1px 黑色轮廓包围整体
```

### 山脚岩石 (约 8×6px)
```
  ████
 ██░░██        ← 岩棕 (#885820) + 深棕轮廓 (#604010)
████████
```
- 散落在山体底部和森林边缘
- 圆润的蘑菇/馒头形状
- 作为山脉与草地的过渡装饰

## 五、森林 (Forest)

### 视觉特征

#### A. 密集森林 (左侧大片区域)
- 由大量**窄尖锥形**树木密集排列组成
- 单棵树极窄：宽约 6-8px，高约 16-24px
- 树木前后层叠，密度极高，几乎看不到树干
- 每棵树有 1px 黑色轮廓
- 树冠颜色：深绿 #186818 为主体，极深绿 #0C400C 为暗面
- 整体效果：形成不可穿行的"绿墙"

#### B. 稀疏树木 (边缘/散落)
- 形状同上但尺寸稍小：宽约 5-6px，高约 10-14px
- 有间距地排列，可见草地底层
- 常出现在森林边缘、山脚作为过渡

### 颜色
| 层 | 颜色 | 用途 |
|----|------|------|
| 轮廓 | #000000 | 1px 黑色描边 |
| 树冠亮面 | #186818 | 朝光面 |
| 树冠暗面 | #0C400C | 背光面 |
| 树干 | #604010 | 仅在稀疏处可见 |

### 单棵树结构 (约 6×20px)
```
   ██           ← 尖端 (#186818)
  ████
 ██████
████████        ← 左暗右亮
 ██████
  ████
 ██████         ← 重复锥形层叠（NES树的特征）
████████
 ██████
  ████
   ██           ← 树干 (#604010)
   ██
```

### 密集森林排列方式
```
树木交错排列，每行偏移 3-4px：
  △ △ △ △ △
 △ △ △ △ △ △
  △ △ △ △ △
 △ △ △ △ △ △    ← 前排遮挡后排
```

## 六、角色精灵 (Character Sprite)

### 视觉特征
- 尺寸：16×16px（不含武器延伸）
- 头部占 5-6px 高，身体占 10-11px 高
- 全身 1px 黑色轮廓
- 头顶装备（蓝色头盔/头巾）突出角色身份
- 身体为简化铠甲：白色基底 + 黑色横条纹
- 武器（剑）为 1px 线延伸到右侧

### 颜色（每个角色限制 4-5 色 + 黑色轮廓）
| 部位 | 颜色 | 说明 |
|------|------|------|
| 轮廓 | #000000 | 全身包围 |
| 头盔 | #3060F0 | 蓝色，3-4px 高 |
| 面部 | #F0B088 | 肤色，2-3px 高 |
| 铠甲 | #F0F0F0 | 白色主体 |
| 铠甲纹 | #000000 | 水平条纹 2-3 条 |
| 武器 | #B8B8B8 | 灰白色，1px 宽 |

### 角色结构示意 (16×16px)
```
     ████         ← 蓝色头盔
    ██████
     ████         ← 肤色面部
    ██░░██
   ██████████     ← 白色铠甲 + 黑条纹
   ██▓▓▓▓██
   ██████████─    ← 武器延伸 (1px)
   ██▓▓▓▓██
    ██  ██        ← 腿部
    ██  ██
```

### 动画帧
- 站立：1帧
- 行走：2帧循环（左脚/右脚交替）
- 每帧仅移动 1-2px 的肢体差异

## 七、关键渲染规则

### 1. 抖动 (Dithering)
所有大面积区域使用 **有序抖动**（ordered dithering），不使用随机噪声：
```
抖动模式 (2×2 Bayer):
  0 2
  3 1

对于双色混合 A/B:
  像素值 < 阈值 → 颜色A
  像素值 >= 阈值 → 颜色B
```

### 2. 轮廓线
- 所有精灵（树、山、角色、岩石）均有 **1px #000000 轮廓**
- 轮廓线不加粗、不模糊
- 地形 Tile（草地、道路）**无轮廓线**

### 3. 层叠遮挡
- 前排对象遮挡后排，通过 y 坐标排序
- 不使用任何透明度/半透明效果
- 遮挡完全不透明

### 4. 缩放
- 所有素材按整数倍缩放（2x, 3x, 4x）
- 禁止分数缩放，保持像素锐利
- `image-rendering: pixelated` / `crisp-edges`

### 5. 禁止使用
- ❌ 渐变 (gradient)
- ❌ 模糊/高斯 (blur)
- ❌ 抗锯齿 (anti-aliasing)
- ❌ 阴影 (drop-shadow / box-shadow)
- ❌ 透明度/半透明 (opacity < 1)
- ❌ 超过 6 色的单个精灵
